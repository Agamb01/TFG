Acronimos

INE - Insituto Nacional de Estadística 

CPU - Unidad Central de proceso.

CPI - Ciclos por instrucción.

PC - Contador de programa.

RISC - Reduced instruction set computer

CISC - Complex instruction set computer

SISC - Simple Instruction Set Computer

IF - Búsqueda de la instrucción

ID - Descodificación de la instrucción 

EX - Ejecución y cálculo de direcciones efectivas

MEM - Acceso a memoria 

WB - Postescritura

RAW - Lectura después de escritura (Read after Write)

WAR - Escritura después de escritura (Write after Read)

WAW - Escritura después de escritura (Write after Write)

RAR - Lectura después de lectura (Read after Read)

SEE - Single-Event Effect

SEU - Single-Event Upset

SET - Single-Event Transient
